![Status](https://img.shields.io/badge/status-active-success)

Provisioned **Rails “Hello World” application** running in Docker, backed by **PostgreSQL** and **Redis**, and fully wired into a **Prometheus + Grafana** monitoring stack with exporters and alerting rules.

> One script (`bootstrap.sh`) takes a clean laptop with Docker and brings up the entire environment.
